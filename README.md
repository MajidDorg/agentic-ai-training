# Agentic AI — Training Course

Welcome! Over this course you'll learn to **design and build AI agents** — systems that reason, use tools, and work together to get real tasks done. You'll learn by building, framework by framework, and finishing each module with your **own** project.

The philosophy: you're being trained to think like an **architect who directs AI** and can **read and judge** what it produces — not to memorize syntax.

---

## 🚀 Start here

1. **Set up your machine** → **[SETUP.md](SETUP.md)** (Mac & Windows; you can let Codex do most of it).
2. Check you're ready any time:
   ```
   uv run check_setup.py
   ```
3. **Begin Module 1** → **[1_openai_agents_sdk/](1_openai_agents_sdk/)** — open `1_lab1_agents.ipynb`.

## How each module works

Three short labs teach the framework. Then **the last lab is yours**: you pick a project, sketch its architecture on paper, let **Codex** build it within what the labs taught, compare your design with Codex's, and finally turn it into a running app with a GUI.
